# Library Management System (Java)

A menu-driven Library Management System developed in Java.

## Features

* Add a book
* Remove a book
* Search for a book
* Display all books
* Count total books
* Prevent duplicate entries

## Concepts Used

* Java
* Arrays / ArrayList
* Methods
* Loops
* Conditional Statements
* Object-Oriented Programming (if applicable)

## How to Run

1. Clone the repository
2. Open the project in VS Code
3. Compile:

javac Library.java

4. Run:

java Library

## Sample Menu

1. Add Book
2. Remove Book
3. Search Book
4. Display Books
5. Count Books
6. Exit
